# Experiment 3.2: CI/CD Pipelines & Automated Testing

## Aim
To implement continuous integration and continuous deployment (CI/CD) pipelines, and build automated testing workflows utilizing GitHub Actions, Jenkins, JUnit, and Mockito.

## Objective
1. **Create CI/CD Pipelines**: Simulate automated workflows running on push/pull_request triggers.
2. **Write Unit Tests**: Demonstrate isolation testing of components using JUnit and Mockito without requiring database connectivity.
3. **Automate Deployments**: Simulate a webhook trigger pushing artifacts to a registry and applying them to a live cluster environment.

## Setup
1. `npm install`
2. `npm run dev`
